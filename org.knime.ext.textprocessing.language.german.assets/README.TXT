It is released under the General Public License (GPL) (see the file 
LICENSE.TXT, which should be located in the same directory as this file). 
If you didn't receive a copy of the license file, please contact us: 
contact@knime.com

This plugin uses models from other projects:

StanfordNLP German POS Models 
- German Dewac (3.6.0)
- German fast (3.6.0)
- German hgc (3.6.0)
https://nlp.stanford.edu/software/tagger.html
StanfordNLP POS tagger is released under the GPL license (http://www.gnu.org/licenses/gpl-2.0.html).


StanfordNLP German NER Models 
- German Dewac (3.6.0)
- German hgc (3.6.0)
https://nlp.stanford.edu/software/tagger.html
StanfordNLP NER tagger is released under the GPL license (http://www.gnu.org/licenses/gpl-2.0.html).
