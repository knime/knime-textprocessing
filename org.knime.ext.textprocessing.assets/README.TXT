This plugin project provides functionality and models for English language as extension for KNIME.

It is released under the General Public License (GPL) (see the file 
LICENSE.TXT, which should be located in the same directory as this file). 
If you didn't receive a copy of the license file, please contact us: 
contact@knime.com

This plugin uses models from other projects:

StanfordNLP English POS Models 
- English 3 Class Distsim (3.6.0)
- English 3 Class No distsim (3.6.0)
- English 4 Class Distsim (3.6.0)
- English 4 Class No distsim (3.6.0)
- English 7 Class Distsim (3.6.0)
- English 7 Class No distsim (3.6.0)
- English 3 Class No wiki caseless distsim (3.7.0)
https://nlp.stanford.edu/software/tagger.html
StanfordNLP POS tagger is released under the GPL license (http://www.gnu.org/licenses/gpl-2.0.html).

StanfordNLP English NER Models 
- English bidirectional distsim (3.6.0)
- English Caseless left 3 words distsim (3.6.0)
- English left 3 words distsim (3.6.0)
https://nlp.stanford.edu/software/tagger.html
StanfordNLP NER tagger is released under the GPL license (http://www.gnu.org/licenses/gpl-2.0.html).
